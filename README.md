# Portfolio Website

Personal portfolio showcasing my experience, projects, and technical skill set.

## Tech Stack
- React
- Vite
- JavaScript / TypeScript
- HTML5
- CSS3
- GitHub Pages

## Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- npm


### To push the changes to the git repo in main branch
Run the following commands in the terminal the pacakge.json had the scripts this command will add all files and commit with a "update" message and push directly to the repo main branch

```bash
npm run git
```

### Local Development
Run the application locally with hot reload:
```bash
npm install
npm run dev
```

### Build & Deployment

Build the production bundle and deploy directly to the portfolio domain:

```bash
npm run build
npm run deploy
```


### or you can use the single cmd to pushed to main branch in the git and also relase the changes to the main domain which is getting the changes from gh-pages with this single cmd

```bash
npm run release
```
