# Portfolio Website

Personal portfolio showcasing my experience, projects, and technical skill set.

## Tech Stack
- React
- Vite
- JavaScript / TypeScript
- HTML5
- CSS3
- GitHub Pages

## Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- npm

### Local Development
Run the application locally with hot reload:
```bash
npm install
npm run dev
```

### Build & Deployment

Build the production bundle and deploy directly to the portfolio domain:

```bash
npm run build
npm run deploy
```
